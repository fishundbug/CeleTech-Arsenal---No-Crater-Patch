# CeleTech Arsenal - No Crater Patch

RimWorld 模组补丁，移除 [CeleTech Arsenal MKIII](https://steamcommunity.com/sharedfiles/filedetails/?id=) 中**红箭50 重型反坦克导弹**（`CMC_Missile_HeavyATGM`）命中后生成弹药坑（`CraterSmall`）的效果。

## 依赖

- [CeleTech Arsenal MKIII](https://steamcommunity.com/sharedfiles/filedetails/?id=3446237098) (`TOT.CeleTech.MKIII`)

## 安装

1. 将本文件夹放入 RimWorld 的 `Mods` 目录
2. 在游戏中启用此模组，确保加载顺序在 CeleTech Arsenal MKIII **之后**

## 兼容性

- 支持 RimWorld 1.5 / 1.6
- 兼容 Combat Extended
